# CIFAR-10 Image Classification using CNN

This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset using PyTorch.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    cd your-repo-name
    ```

2. Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. To train the model:
    ```bash
    python main.py
    ```

## Project Structure

- `main.py`: The main script for training, validating, and testing the model.
- `config.py`: Configuration parameters for the model and training.
- `requirements.txt`: List of required Python packages.
- `README.md`: Project documentation.
